# 타자기 프로젝트 (Typing Machine)

## 📋 제작자 정보

- 학번: 2402148
- 소속: 한양여자대학교

## 🛠 기술 스택
- HTML, CSS, JavaScript

## 🧩 기능
- 키보드 형태의 버튼 클릭 시 텍스트 출력
- 출력은 label로 구현된 종이에 표시
- 색상별 그룹으로 키 구분

## 📂 파일 구성
- `index.html`: 메인 웹 페이지
- `style.css`: 키보드 및 종이 스타일
- `script.js`: 타자기 동작 로직
